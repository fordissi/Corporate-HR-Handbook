# 企業人事管理手冊 (Corporate HR Handbook)

本儲存庫存放公司所有符合 ISO 規範之 HR 管理文件。

## 文件階層
- **第一階 (MN)**: [管理手冊](./Employee%20Handbook/HR-MN-QM-01_員工管理手冊.md)
- **第二階 (PR)**: 管理程序 (考勤、招募、績效、薪資等)
- **第三階 (WI)**: 作業指導書 (系統操作指南)
- **第四階 (FM)**: 表單附件

## 維護規範
1. 所有修改必須透過 **Commit** 記錄原因。
2. 涉及法律條文修改，請標註 **Version** 與 **核准主管**。
