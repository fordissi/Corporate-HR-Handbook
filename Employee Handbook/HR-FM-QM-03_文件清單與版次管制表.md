## 分類原則

本清單作為員工管理手冊相關文件之現行管制索引。文件仍維持原文件編號與檔名，不另以改名或搬移資料夾區分狀態，以避免既有連結、文件引用及版本追蹤失效。

文件狀態以各文件 metadata 的 `status` 欄位及本清單同步管制：

| 狀態 | 定義 |
| --- | --- |
| draft | 草稿，尚未完成審核或尚未決定發行 |
| review | 審核中 |
| approved | 已核准，待公告或指定生效日 |
| active | 已審核、現行執行或已可發行 |
| archived | 已停止適用，僅供歷史追溯 |

## 管制提醒
