## 一、 目的
為規範員工因公出差之申請、費用報支標準及核銷作業，特訂定本程序。

## 二、 出差申請
1. 員工因公需前往外地處理業務，應於出差日前透過公司指定系統或行動 App 填寫公出差旅單並完成簽核（現行為 104 App）。如系統或行動 App 無法使用，始依人事課或財會課指定之備援表單辦理。
2. 國內出差由部門主管核決；國外出差需呈報至 **總經理** 核准。
3. 若申請系統僅提供自由敘述欄位，員工應於敘述內容中完整載明出差目的、行程、時間及預估費用，以利主管判斷必要性及財會後續報支。

### 2.1 申請內容應填資訊

出差申請至少應包含下列資訊：

| 項目 | 填寫內容 |
| --- | --- |
| 出差目的 | 說明出差原因，例如客戶拜訪、會議、教育訓練、招標、產品推廣、供應商或跨部門協調。 |
| 出差對象 | 填寫拜訪客戶、合作單位、會議主辦單位或內部接洽窗口。 |
| 出差地點 | 填寫縣市、地址或主要活動地點；多地點行程應分別列明。 |
| 預計出發時間 | 填寫預計從公司、住處或主管核准地點出發之日期及時間。 |
| 預計活動時間 | 填寫公務活動開始及結束時間，作為膳雜費半日或全日判斷依據。 |
| 預計返回時間 | 填寫預計返回公司、住處或主管核准地點之日期及時間。 |
| 交通方式 | 填寫高鐵、台鐵、客運、捷運、計程車、自行開車或其他方式。 |
| 是否住宿 | 如需住宿，應填寫住宿日期、地點、預估金額及是否符合住宿限額。 |
| 預估費用 | 填寫交通、住宿、膳雜費及其他必要費用之預估金額。 |
| 預期成果 | 說明預期完成事項，例如完成拜訪、取得資料、完成協調、提交報告或帶回待辦事項。 |

自由敘述範例：

```text
出差目的：參加台中市政府採購案投標前說明會，確認標案規格、履約條件及投標文件要求。
出差對象：台中市政府採購承辦單位及標案說明會窗口。
出差地點：台中市政府市政大樓。
時間：2026/07/15 08:00 自台北出發，10:30-12:00 參加標案說明會，預計 15:30 返回台北。
交通方式：高鐵往返，市區以捷運或計程車銜接。
住宿：無。
預估費用：高鐵 1,400 元、當地交通約 300 元、膳雜費半日 250 元。
預期成果：取得標案說明會資料，確認投標資格、文件清單、送件期限及需內部補充之資料。
```

## 四、 交通費用標準
1. **大眾運輸**：以搭乘高鐵（標準廂）、台鐵或客運為原則。
2. **計程車/叫車服務**：限緊急公務、攜帶重物或偏遠地區無大眾運輸時使用，核銷時需註明起訖點。
3. **自行開車**：依公司公告之公里數補貼標準報支油資與過路費。

## 五、 住宿費用限額
出差地點若需過夜，每晚住宿限額如下（實報實銷）：
- **六都（台北市、新北市、桃園市、台中市、台南市、高雄市）**：每晚最高 NT$ 2,000。
- **其他縣市**：每晚最高 NT$ 1,500。
- 住宿費不分職級，以出差地點所在縣市認定。
- 若因特殊地區房價、展會期間、臨時任務或安全因素致無法於限額內安排住宿，應事前敘明原因並經主管核准；未事前核准者，超出限額部分原則上不予報支。

## 六、 膳雜費補貼
膳雜費原則上適用於跨縣市出差。員工於公司所在地同縣市內辦理短時間公務外出，例如至主管機關參加政令宣導、洽公或遞送文件者，原則上不給予膳雜費，僅得依規定核實報支必要交通費。

同縣市公出如因公司指派長時間參與活動、跨越主要用餐時段且公司或主辦單位未供餐，得由主管視實際情形專案核准餐費或膳雜費。

符合膳雜費適用條件之國內出差標準如下：
- **半日出差**：NT$ 250。
- **全日出差**：NT$ 500。
- **國外出差**：依目的地、行程及核准內容另行簽核。

### 6.1 半日與全日判斷

膳雜費以出差活動時間加計預估必要往返交通時間綜合判斷：

| 類型 | 判斷原則 | 補貼金額 |
| --- | --- | --- |
| 市內公出 | 公司所在地同縣市內之短時間公務外出，例如參加政令宣導、洽公或遞送文件。 | 原則上不給予膳雜費，僅核實報支必要交通費 |
| 半日出差 | 跨縣市出差，且出差活動連同預估必要往返交通時間未達 8 小時。 | NT$ 250 |
| 全日出差 | 跨縣市出差，且出差活動連同預估必要往返交通時間達 8 小時以上。 | NT$ 500 |

判斷時應以實際行程為準，包含從公司、住處或主管核准地點出發之時間、出差活動起迄時間及返回時間。若公司、客戶或活動主辦單位已供應餐食，主管或財會課得依實際情形調整或不予補貼該餐相關費用。

### 6.2 膳雜費申請流程

1. 員工於出差申請時，應於申請內容載明預計出發時間、出差活動時間、返回時間及預估半日或全日膳雜費。
2. 出差結束後，員工應於《差旅費報支單》(HR-FM-TRV-01) 填列實際出發、活動及返回時間，並勾選半日或全日補貼。
3. 直屬主管應依實際公務行程確認半日或全日判斷是否合理。
4. 財會課依核准之報支單辦理撥款；如行程時間、活動內容或餐食供應狀況不明，得請員工補充說明。
5. 膳雜費採定額補貼，原則上免檢附餐費發票；但交通、住宿及其他實報實銷項目仍應依規定檢附憑證。

## 七、 報支程序
1. **時限**：出差結束後 **7 個工作日** 內，應整理單據並填寫《差旅費報支單》(HR-FM-TRV-01) 及《差旅報告書》(HR-FM-TRV-02)。
2. **憑證**：除膳雜費採定額補貼外，其餘交通與住宿均需檢附正式發票或收據。

### 現行SOP內容

### 目的 Purpose

為規範本公司員工因公務出差之差旅費用報支作業，確保費用使用合理、透明，並符合稅務及勞動法規要求，特訂定本辦法。本辦法旨在促進業務效率，同時防範虛報或不當使用公司資源，維護公司財務合規性。

### 適用範圍 Scope

本辦法適用於本公司全體員工（含正式員工及兼職員工）因公務出差之交通、住宿及膳雜費報支事宜。出差多為客戶拜訪、配合招標作業、產品推廣、會議參與或供應鏈協調等業務需求。

### 相關文件 Related Documents

### 定義 Definitions and Abbreviations

出差：指員工離開公司所在地（台北市）執行公務，包含國內出差及國外出差。

國內出差：出差地點為台灣本島、澎湖、金門、馬祖等地區。

國外出差：出差地點為台灣以外地區（含中國大陸、港澳及海外）。

短程出差：出差地點為公司所在地縣市內或當日可往返者（不需住宿）。

長程出差：出差地點需跨縣市或需住宿者。

差旅費用：包含交通費、住宿費及膳雜費，原則上採實報實銷，但不得超過本辦法規定上限。

### 內容 Process Description

#### 法遵原則

- 所有差旅費用須檢附合法憑證（如統一發票、收據），並符合財政部稅務規定。虛報或無憑證者視為違規，可能扣除薪資或列入紀律處分。

- 出差期間視同正常上班，員工應遵守勞動基準法相關規定（如加班補休）。若涉及國外出差，須注意出入境法規及醫療產品進出口合規。

- 公司保留審核權，若發現不當使用（如個人消費混入公務），將追償並通報主管機關。可核銷項目

#### 出差申請

員工出差前，須填寫「出差申請單」（附件一），詳細說明出差目的、地點、日期、預估費用及預期成果，經直屬主管及部門經理核准。國外出差須呈報總經理核准。

短程出差可於出差當日申請公假或公出，長程出差至少提前3日申請。

出差期間若需延長，須即時以電話或電子郵件請示主管，並於返回後補辦手續。未經核准延長部分不予報支費用。若發票及收據無消費明細內容則需另外檢附商家之產品資訊，如報價單、菜單、產品價目表等文件佐以證明。

如為說明會類型活動（例如Seminar、RTD、Detailing meals等）則須檢附相關紀錄（如照片、參與人名單或會議議程）。

#### 出差報告

出差返回公司後，須填寫「出差費用申請單」（附件二），包含實際行程、成果及費用明細，呈報主管審核。未提交報告者，不得報支費用。

#### 預借條款

長程出差可於出差前申請預借款項，填寫「預支費用申請單」（附件三），經財務部核准後領取。返回後須於7日內結算，多退少補。

#### 差旅費用標準

#### 交通費：

原則：以大眾運輸工具為主，鼓勵環保及成本控制。實報實銷，但須檢附憑證（如車票、發票）。

#### 標準：

高鐵/火車/長途巴士：實報實銷。

市內公車/捷運：無憑證者，可按實際起訖點申報，每日上限NTD 200。

私車公用：補助油資每公里NTD 6-8（依中油95無鉛汽油公告價調整），停車費及過路費實報實銷（檢附證明）。多人共乘僅補助一人。

計程車：僅限必要公務，每趟上限NTD 500，須註明事由。

法遵考量：私車公用須確保車輛保險有效，避免事故責任歸屬爭議。

#### 住宿費

原則：選擇安全、衛生之住宿場所，優先考慮與醫療客戶相關之商務旅館。實報實銷，但不得超過上限。

#### 標準（單人房，每晚上限）：

台北/新北/桃園/台中/台南/高雄：NTD 2,000。

其他縣市：NTD 1,500。

國外出差：依目的地調整，須事先核准。

若住宿客戶提供之宿舍或親友處，補助NTD 800/晚，但須檢附等額加油發票或相關證明，否則列入個人所得稅申報。

法遵考量：住宿費須開立公司統一編號發票，確保稅務扣抵合規。

#### 膳雜費

原則：涵蓋膳食及雜支（如通訊、影印），按日計算。出差當日若上午出發或下午返回，調整為2/3；下午出發或上午返回，調整為1/3。

#### 標準（每日）：

主管級（經理以上）：NTD 600（早餐NTD 100、午餐NTD 200、晚餐NTD 200、雜費NTD 100）。

一般員工：NTD 500（早餐NTD 80、午餐NTD 180、晚餐NTD 180、雜費NTD 60）。

若公司或客戶供應餐食，不予報支該餐。

短程出差：僅報支膳雜費NTD 200/日（不含住宿）。

國外出差：每日NTD 1,000-1,500，依目的地調整。

法遵考量：膳雜費無憑證者，按定額報支，但超過NTD 3,000/月部分須列入個人所得稅計算。

#### 報銷程序

#### 報銷申請

出差返回後7日內，填寫「差旅費用報支單」（附件四），檢附憑證及報告單，經主管審核後送財務部。

財務部於每月薪資結算前審核，逾期不予受理。

多退少補原則：預借款項結算後，若有餘額須立即退還公司。

#### 審核與追責

主管及財務部負責審核憑證真實性，若發現不實，員工須承擔責任（含扣款或解僱）。

年度差旅費用超過預算10%者，須向總經理報告並調整。

#### 附則

本辦法自公告之日起施行，修訂時經總經理核准後公告。公司保留解釋權，若與法規衝突，以法規為準。
